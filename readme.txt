Программа учета граждан, встающие в качестве безработных в ЦЗН.

Приложение развернуто на сервере.
Уровни доступа:
- Посетитель
    Описание:   при заходе на сайт видит стартовую страницу центра занятости, возможную документацию для изучения, статьи,
                контакты.
                Посетитель может подать онлайн заявление в качестве безработного, его получает сотрудник ЦЗН.
                При отправке заявления, посетителю приходит сообщение на почту о статусе регистрации, а также последующие
                автоматические сообщения с напоминанием о необходимости отметки в ЦЗН.
                При регистрации посетителю приходит пароль на почту, чтобы он мог просмотреть на сайте ЦЗН о
                своем статусе и возможностях.

- Сотрудник ЦЗН
    Описание:   сотрудник имеет доступ к закрытой части программы, для регистрации безработных

- Админ
    Описание:   У админа есть доступ к странице по добавлению новых пользователей с ролью "Работник ЦЗН".
