Программа учета граждан, встающие в качестве безработных в ЦЗН.

Приложение развернуто на сервере.
Уровни доступа:
- Посетитель
- Сотрудник ЦЗН

1 этап.
- Создание моделей.
- Подключение БД. Создание таблиц. Связка моделей с БД
- Создание логики
- Подключение MVC. Создание стартовой страницы, создание форм ввода данных гражданина
